# Yoka's Blog

Personal blog built with HTML/CSS, hosted on GitHub Pages.

## Structure

```
yoka.github.io/
├── index.html          # Home page
├── about.html          # About me
├── gallery.html        # Gallery / Projects
└── posts/
    └── hello-world.html  # Blog posts go here
```

## Adding a new post

1. Copy `posts/hello-world.html` and rename it
2. Edit the content inside
3. Add a link to the new post in `index.html`
